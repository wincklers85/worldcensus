Come deve funzionare per davvero

Per avere ricerche reali senza upload manuale, il progetto va alimentato con dataset ufficiali preindicizzati oppure con API pubbliche dove esistono. Quando un paese non offre un endpoint aperto dal browser, il sistema usa snapshot JSON aggiornati e pubblicati nel repository.

Perché non puoi avere tutto live dal browser

Molti enti non espongono API aperte per modello/versione/colori, oppure impongono limiti, autenticazione o restrizioni CORS. Quindi la soluzione seria è una ricerca federata con adattatori per paese, non una promessa finta.
